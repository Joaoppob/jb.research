# Meu Projeto Web

Modelo inicial de estrutura de projeto para hospedagem no **GitHub Pages**.

## Estrutura de Pastas e Arquivos

- `index.html`: Arquivo principal da página, que será lido por padrão pelo GitHub Pages.
- `css/`: Pasta para folhas de estilo (como `style.css`).
- `js/`: Pasta para scripts (como `script.js`).
- `assets/`: Pasta para armazenar mídias, como imagens (`assets/images/`), fontes, etc.

## Como hospedar no GitHub Pages

1. Faça o commit e push desses arquivos para um repositório no GitHub.
2. No seu repositório, vá em **Settings** > **Pages**.
3. Em "Source", selecione onde sua branch principal está localizada (normalmente `main` ou `master` no diretório `/root`).
4. Clique em "Save". Em alguns minutos, seu site ficará disponível através da URL fornecida pelo GitHub.
